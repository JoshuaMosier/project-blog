Introducing Plotify, a lightweight JavaScript library for creating responsive and interactive data visualizations. Built with performance and ease of use in mind.

![Plotify Demo](./2021_Oct-20_plotify.png)

## Features
- Declarative API
- Responsive design
- Custom themes
- Animation support
